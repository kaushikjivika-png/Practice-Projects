# 📰 Fake News Headline Generator

A fun and beginner-friendly Python project that generates random fake news headlines by combining random subjects, actions, and places. The project uses Python's built-in `random` module to create humorous and unexpected news headlines.

## 🚀 Features

* Generates random fake news headlines
* Interactive command-line interface
* Unlimited headline generation until the user exits
* Beginner-friendly Python project
* Uses Python's built-in `random` module

## 🛠️ Technologies Used

* Python 3
* Random Module
